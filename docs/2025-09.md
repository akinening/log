# 2025.09まで

**主要成果**

- **複数プロダクトのデザイン推進と体験価値向上**
    
    Goghモバイル・Steam・Robloxといった複数プロダクトにおいてUI/UXデザインを主導。
    
    各プラットフォーム特性に合わせた最適化を行い、ambrが提供する体験価値向上に大きく貢献。
    
- **新規事業の立ち上げと共通基盤デザイン構築**
    
    新規事業のアイデア創出や改善施策を積極的に提案。
    
    Roblox向けには共通UI基盤を設計し、複数タイトル間で再利用可能なデザインシステムを整備。
    
    これにより、今後の開発効率およびスピードを大幅に向上させる基盤を確立。
    
- **課金フロー設計および販売体験の最適化**
    
    プロトタイプを活用して課金仕様・販売方法を検討。
    
    Steam版ではデスクトップに最適化された体験設計を行い、新たなユーザー層の獲得に成功。
    
- **法務・運用・開発との連携によるリスクマネジメントと改善**
    
    法務・運用担当（田中氏・松田氏）との連携を通じ、課金に関する懸念点を整理・解消。
    
    エンジニアチームとの協働でオンボーディングおよび課金フローを改善し、ユーザー離脱率の低減に寄与。
    
- **AI活用によるデザインプロセスの効率化**
    
    生成AIを積極的に導入し、アイデア具現化までのリードタイムを大幅に短縮。
    
    試作～検証サイクルを高速化し、開発全体の生産性向上に貢献。
    
- **リサーチに基づくUX改善提案**
    
    GoghやRobloxの仕様検討にあたり、国内外の類似アプリやエクスペリエンスをリサーチ。
    
    他社事例を踏まえた改善提案を行い、より洗練されたユーザー体験を実現。
