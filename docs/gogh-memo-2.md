

# PMF前ピボットの経緯

## ユーザーを観察した結果判明したこと

- 当初は面白いアバターやアニメシーン再現などの遊びを考えていた
- 実際には、エモい部屋の写真・動画が多く投稿されていた

## 既存メタバースの課題点

- メタバースの課題は、その空間に行く動機が薄いことと、アクセスまでのハードルが高いこと
    - その空間に入ってやることがない。
    - 知らない人と話したくない。
- VRChatやClusterの課題は、誰がいるかわからず、そもそも入って良いかもわからないこと
- 誰もいない空間ほど虚無なものはない。ただ空間で喋れるようにしただけでは使われない。
    - テンプレのバーチャル部屋があったとしても、そこに人を呼びたいとは思わない。ボイチャでいいので。

### Not For meな空間

- 部屋がたくさん並んでいても、見知らぬ空間、見知らぬアバターにそこまで興味がない。数回出入りしたら飽きる。
- RobloxやZEPETOのように、ゲームをする空間がたくさんあるのは動機が明確。
    - 圧倒的な訪問者数で、動きやテキストで盛り上がれる。
    - ただし誰と話したか覚えておらず、深い関係性は構築できない。

## 場に目的を持たせることが重要

- 話題ベースでつながる。たとえばオタトーク・オフ会としての雰囲気作りが必要
- オープンな場合は、相手が信頼できるか判断する「前フリ」が必要そう
